Guia Completo: Firebase Authentication

Authenticação no Firebase

Configuração passo a passo

1. Aceder á consola do projeto hosted em firebase

Entrar no site do firebase e entrar na conta onde o projeto está registado.
Após isto deve clicar na aba de Authentication (Ou em "Todos os Serviços" caso não apareça).
Asseguir clicar em "Vamos Começar" para iniciar o processo.

2. Escolher o Provider

Selecionar o Google como provider

3. Configurar o gmail ligado á autenticação

Ativar o provider no botão

E depois clicar na drop-down box e selecionar o email mostrado. Caso exista vários, escolha um.
Também deve dar um nome relacionado ao projeto para ficar mais organizado

4. Finalizado

Com isto, o seguinte menu deve aparecer, a mostrar que a Autenticação com o Google está ativada
